React with Redux

Front End Development Libraries Live web development
  • typically a single redux store will be used for an entire react application
  • a special library is needed to integrate redux with react (react-redux)
  • react-redux lib comes with Provider wrapper which wraps the react app in order to expose Redux state to react app
  • access to the state is provided to redux via the mapStateToProps and mapDispatchToProps methods
  • mapDispatchToProps method exposes action creators to a specific react component
